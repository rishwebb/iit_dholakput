## Quick Look

IIT Dholakpur is a humor-filled fictional institute website that mixes polished visuals with absurd campus lore.  
From **emergency laddoo alerts** to **anti-gravity mode**, the project leans fully into the joke while still feeling like a complete, modern static site.

### Highlights

| Feature | What’s inside |
| --- | --- |
| 🎨 Beautiful visual style | A bold dark theme, clean layout, premium typography, and polished cards across every page |
| 🌗 Theme toggle | Light/dark mode support with theme preference saved in `localStorage` |
| 🧆 Interactive fun | Laddu clicker, emergency laddu drop, animated counters, anti-gravity mode, and modal interactions |
| 🏛️ Multi-page experience | Dedicated pages for Academics, Admissions, Placements, Campus Life, and Faculty & Staff |
| 😄 Full parody energy | Dholakpur-inspired writing, outrageous faculty bios, and legendary campus storytelling |

## Local Preview

Because this is a static site, you can preview it locally in seconds:

```bash
cd iit_dholakput
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000/`

You can also open `index.html` directly in a browser, but using a simple local server is the cleanest option.
